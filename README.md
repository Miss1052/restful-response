一个专业的Laravel扩展包，用于统一RESTful API响应格式，采用接口模式设计，提供标准化的JSON响应格式。

## 特性

- 🎯 **统一响应格式** - 标准化的JSON响应结构
- 🔌 **接口驱动** - 基于接口的可扩展设计
- 🛡️ **异常处理** - 自动处理常见异常并格式化响应
- 🌐 **多语言支持** - 内置中英文语言包
- 🧪 **完整测试** - 100%代码覆盖率
- 📦 **开箱即用** - 零配置快速开始
- 🎨 **灵活配置** - 可自定义响应格式和字段名称
